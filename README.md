# Foobartory
Clone the repository and move into the project to install dependencies and launch the game.


## Installation

Use your package manager to install dependencies

```bash
yarn install
```
or

```bash
npm install
```

## Usage

```bash
yarn start
```
or 

```bash
npm start
```


## Tests

```bash
yarn test
```
or 

```bash
npm test
```
